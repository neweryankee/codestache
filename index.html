Cucumber Workshop

Vermonster and Plectix are hosting a free two hour workshop focusing on everyone's favorite new tool, Cucumber.

Grab your computer and come see what all the noise is about.

We'll give you a quick intro, do some hand-holding, and leave you to some pair exercises.

Food and beverages will be on the house.

Hopefully this will serve as the first of many such events in a series I call "The Newer Yankee Workshops" for New Englanders who want to get there Ruby on.

Recent versions of Git, Ruby and Gem required. Toolbelt and flannel optional.

Prerequisites

  • OSX, BSD or Linux
  • Git
  • Ruby 1.8.x
  • Rubygems
  • Need to be able to compile C libraries (Xcode, build-essentials, etc.)

If you need help with this, or are a Windows user, checkout out this guide.

Setup

Install Gemcutter Gem


      $ gem install gemcutter
      $ gem tumble

Install Gem Bundler


      $ gem install bundler

Clone repository


      $ git clone git://github.com/neweryankee/codestache.git

Gem bundle


      $ cd codestache
      $ gem bundle

Test it out

Make sure things are green...


      $ rake
